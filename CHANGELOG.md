# Changelog

## [0.2.0] - 2025-11-16

### Added

- Configuration file for defining supported language file/test patterns, ignore patterns, etc.

### Fixed

- Fixes [#1](https://github.com/kevinbraun/vscode-hop-to-test/issues/1) - Unable to find source file for test file with multiple patterns

### Added

- Initial release
- Command to hop between source files and their related test files - specifically for JS/TS for the time being
- Default keyboard shortcut (Ctrl+Shift+T / Cmd+Shift+T) to trigger the hop command
